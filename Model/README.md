# Model

Model code, training scripts, and saved checkpoints.

## Suggested layout

| Subfolder | Purpose |
| --- | --- |
| `src/` | Model architecture and training code |
| `configs/` | Hyperparameters and run configurations |
| `checkpoints/` | Saved model weights |
| `notebooks/` | Experiment and exploration notebooks |

## Notes

- Record the training data version and config used for each checkpoint.
- Keep large checkpoint files out of version control.
