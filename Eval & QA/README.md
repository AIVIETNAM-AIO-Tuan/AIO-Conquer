# Eval & QA

Evaluation scripts, benchmarks, and quality-assurance checks.

## Suggested layout

| Subfolder | Purpose |
| --- | --- |
| `metrics/` | Scoring and metric definitions |
| `benchmarks/` | Benchmark datasets and harnesses |
| `results/` | Raw evaluation outputs |
| `qa/` | Quality-assurance checks and validation tests |
